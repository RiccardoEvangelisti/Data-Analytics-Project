*Project purpose*: to recognize the **year** in which a song was published based on the features of its audio track.

To carry out a data analytics study we implemented all the analytical pipeline:
- Data Acquisition
- Data Visualization
- Data Preprocessing 
- Modeling with 
    - ***Linear Regression***
    - ***Random Forest***
    - ***K-Nearest Neighbors***
    - ***Support Vector Regressor***
    - ***Deep Neural Network***
    - ***TabNet***
    - ***TabTransformer***
- Performance Evaluation

The csv file can be found [https://www.kaggle.com/datasets/gauravduttakiit/million-song-dataset/data](here)
